# Archive Folder

This folder contains old, unused, or development-related files that are not required for running the main NoApplAI application.

## Contents

### `flask-sqlalchemy-project/`
A separate Flask-based demo project used for learning SQLAlchemy. Not part of the main NoApplAI application.

### `backup/`
Old HTML page versions from development (Page_1 through Page_7, Full_page). Kept for reference.

### `Extra_Data/`
Contains bookmarks and other miscellaneous data files not needed for the application.

### Development Artifacts
- `footer-html-css.html` - Standalone footer component HTML/CSS for testing
- `FooterComponent.jsx` - React footer component (not used in main app)
- `test_api.html` - HTML file for testing API endpoints
- `test_programs_api.py` - Python script for testing programs API

**Note**: These files are archived and not maintained. They are kept for historical reference only.
